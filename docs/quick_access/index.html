Quick access Menu

  • Whois Lookup: Provides domain information lookup service, allowing users to retrieve registration and ownership details for domains.

  • Nslookup: Offers online tools for DNS lookup, providing information about DNS records, IP addresses, and domain names.

  • Shodan: a search engine that lets users find specific types of computers (webcams, routers, servers, etc.) connected to the internet using a variety of filters, allowingS to discover potentially vulnerable devices and networks.

  • DNS Dumpster: A tool for discovering DNS information about a domain. It provides details such as subdomains, DNS records, and DNS history, helping users gain insights into the domain’s infrastructure and potential attack surface.

  • URLScan.io: A website and file scanning service that helps detect and analyze suspicious URLs and files, provides insights into the behavior of web pages, identifying potential threats such as phishing attempts, malware distribution, and other malicious activities.

  • Cisco Talos: Talos offers comprehensive threat intelligence, including real-time updates on emerging threats, advanced security research, and analysis to help organizations stay ahead of cyber adversaries.

  • VirusTotal: Offers free online file and URL analysis for detecting malware by scanning files and URLs against multiple antivirus engines and threat intelligence sources.

  • MITRE ATT&CK: A globally-accessible knowledge base that provides a comprehensive framework for understanding and categorizing the various tactics and techniques used by cyber adversaries during different stages of an attack, exlained in Level 3

  • CyberChef: An open-source web application tool, used for performing various cryptographic operations including encoding, decoding, hashing, compression, and more.

  • Cryptii: An online tool for encoding, decoding, and converting text using various cryptographic algorithms and encoding methods, supports a wide range of ciphers, including Caesar cipher, Morse code, Base64, Binary, and many others.

  • Rumkin Cryptography Tools: A collection of online tools for encrypting and decrypting text using classic and modern cryptographic algorithms, offers a wide range of ciphers, including Caesar cipher, VigenÃ¨re cipher, Playfair cipher, and more, In addition to encryption and decryption, it provide options for text manipulation, frequency analysis, and encoding conversion.

  • Hybrid Analysis: Provides malware analysis reports and sandbox execution for suspicious files, offering insights into malware behavior and characteristics.

  • MetaDefender: Offers comprehensive threat intelligence and multi-scanning capabilities, allowing users to analyze files, URLs, and IP addresses for potential security risks using multiple antivirus engines and data sources.

  • BuiltWith: Allows users to discover what websites are built with by looking up technology profiles, including web servers, analytics tools, and many more.

  • GTmetrix: Analyzes the loading speed of web pages and provides recommendations for improving performance. It offers insights into how well a website performs and how it can be optimized.

  • Have I Been Pwned: Checks if an email or phone has been compromised in a data breach.
